How to install Coq + Emacs + Proof General on Windows

Coq

  1. Download Coq from https://coq.inria.fr/download, and install it. As an example, my installation directory is C:\Coq.
  2. Add the installation directory (in my case C:\Coq\bin) to the system environment PATH. Here is what is and how to set PATH.

Emacs

  1. Download Emacs for Windows from https://www.gnu.org/software/emacs/download.html#windows.
  2. Run emacs after installation, and do some random change of options in the "Options" menu (you can always change it back) an then press "Save options". This will generate .emacs in you HOME directory. For me on Windows 10, as an example, the generated .emacs is found at C:\users\userID\AppData\Roaming.
  3. Add the following lines to the .emacs file to use the MELPA repository for installing Proof General and others later.
    (require 'package)
    (add-to-list 'package-archives '("melpa" . "https://melpa.org/packages/") t)
    (package-initialize)
  4. Here are more detailed instructions for installing Emacs.

Proof General

  1. You may install and byte-complie Proof General, Company-Coq (a collection of extensions for Proof General's Coq mode, which is optional), and its dependencies by typing successively:
    M-x package-refresh-contents RET
    M-x package-install RET proof-general RET
    M-x package-install RET company-coq RET
    By default, M-x is hitting ALT and X simultaneously, and RET is hitting ENTER.
  2. Add the following to your .emacs file to enable Company-Coq automatically:
    (add-hook 'coq-mode-hook #'company-coq-mode)
  3. More detailed instructions can be found here and here.
